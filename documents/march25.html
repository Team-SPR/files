Further Information Regarding SPRDZY Outage - 06/03/2025

Hello again,

After previously addressing you via the SPRDZY Discord server, we have decided to turn it into a live webpage to update those who are not in the support server.

SPRDZY is made with a special piece of software called Discord Bot Builder (DBB for short). It helps compile the JavaScript code (which is the stuff used to make bots) and helps us run it. The developers of that software have released a broken update, corrupting the code of SPRDZY. While we have lost no data (as far as we know), we cannot restore hosting or development until a patch is produced.

LIVE:
Manager of DBB has announced hopes for fixes this week
